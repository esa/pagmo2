List of problems (UDPs) available in pagmo/pygmo¶

This is the list of user defined problems (UDPs) currently provided with pagmo/pygmo. These are classes that can be used to construct a pagmo::problem, or pygmo.problem which will then provide a unified interface to acces their functionalities.

Each problem can be can be of type S = Single, M = Multi-objective, I = has an integer part, C = Constrained, U = Unconstrained, sto = stochastic

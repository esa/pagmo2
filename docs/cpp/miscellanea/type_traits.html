Type traits and enums¶

Type traits and enums used in PaGMO

enum pagmo::thread_safety¶

Thread safety levels.

This enum defines a set of values that can be used to specify the thread safety of problems, algorithms, etc.

Values:

none¶

No thread safety: any concurrent operation on distinct instances is unsafe.

basic¶

Basic thread safety: any concurrent operation on distinct instances is safe.

template <typename T>
class pagmo::is_udp¶

Detect user-defined problems (UDP).

This type trait will be true if T is not cv/reference qualified, it is destructible, default, copy and move constructible, and if it satisfies the pagmo::has_fitness and pagmo::has_bounds type traits.

Types satisfying this type trait can be used as user-defined problems (UDP) in pagmo::problem.

const bool value = implementation_defined¶

Value of the type trait.

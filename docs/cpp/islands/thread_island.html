Thread island¶

class thread_island¶

Thread island.

This class is a user-defined island (UDI) that will run evolutions directly inside the separate thread of execution within pagmo::island.

Public Functions

std::string get_name() const¶

Island’s name.

Return
"Thread island".

void run_evolve(island &isl) const¶

Run evolve.

This method will use copies of isl’s algorithm and population, obtained via island::get_algorithm() and island::get_population(), to evolve the input island’s population. The evolved population will be assigned to isl using island::set_population().

Parameters
  • isl: the pagmo::island that will undergo evolution.
Exceptions
  • std::invalid_argument: if isl’s algorithm or problem do not provide at least the pagmo::thread_safety::basic thread safety guarantee.
  • unspecified: any exception thrown by island::get_algorithm(), island::get_population(), island::set_population().
